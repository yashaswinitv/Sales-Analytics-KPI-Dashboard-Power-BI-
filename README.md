# 📊 Sales Analytics & KPI Dashboard (Power BI)

## 🚀 Project Overview
This project is an **end-to-end Sales Analytics Dashboard built using Power BI** to analyze business performance across sales, customers, products, and regions.  

It transforms raw sales data into meaningful insights using **data modeling, DAX calculations, and interactive visual storytelling**.

The goal is to enable **data-driven decision-making** by tracking key business KPIs and uncovering trends in sales performance.

## 🎯 Business Objectives
- Monitor overall sales performance in real time  
- Identify top-performing and underperforming products  
- Analyze regional and category-wise sales trends  
- Track customer purchasing behavior  
- Improve business decisions using data insights  

---

## 📊 Key Performance Indicators (KPIs)

- 💰 Total Revenue  
- 📦 Total Quantity Sold  
- 📈 Total Profit  
- 👥 Total Customers  
- 🧾 Average Order Value (AOV)  
- 🏆 Top Products by Sales  

---

## 🧠 Key Insights Derived
- Identified top revenue-generating product categories  
- Discovered seasonal trends impacting sales performance  
- Highlighted regions with highest and lowest sales contribution  
- Found high-value customer segments driving major revenue  
- Observed product demand patterns across categories  

---

## 🧰 Tools & Technologies Used

- Power BI Desktop – Data visualization & dashboard building  
- Power Query Editor – Data cleaning & transformation  
- DAX (Data Analysis Expressions) – KPI calculations & measures  
- Excel / CSV – Data source  

---

## 🔄 Data Workflow

1. Data Collection – Imported raw sales dataset  
2. Data Cleaning – Handled missing values, duplicates, formatting issues  
3. Data Transformation – Structured data using Power Query  
4. Data Modeling – Built relationships between tables  
5. DAX Calculations – Created KPIs and measures  
6. Visualization – Designed interactive dashboard  
7. Insights Generation – Extracted business insights  

---

## 📊 Dashboard Features

- Interactive slicers (Region, Category, Date)  
- Dynamic KPI cards  
- Time-series sales analysis  
- Category-wise performance breakdown  
- Region-wise revenue comparison  
- Top N analysis (Products & Customers)  
- Drill-down enabled visuals
